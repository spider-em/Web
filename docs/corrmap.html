Correspondence map

Displays a correspondence map containing numerical labels for images on the screen. The user can then draw a polygon enclosing a group of image numbers. The average of the group of images contained within the polygon can be displayed, and the average image can stored in a file. The average image display can be temporary or permanent. Additionally a list of the image numbers in the group can be saved to a document file.

Input for this command:

document file name
file contains a list of image numbers (as document file keys). The document file is created by converting the IMC*** file (from correspondance analysis) to a document format using the SPIDER operation : SD C .

Image file name template
Template for the images whose numbers are listed in the document file

First image key
First image number included in map

Last image key
Last image number included in map. (This allows skipping some images listed in the document file).

X axis factor register
Register in document file which contains factor to be plotted on X axis.

Y axis factor register
Register in document file which contains factor to be plotted on Y axis.

Extent (std. dev.)
The extent of outlying image locations which are included in the displayed map.

Lower image Threshold
Upper image Threshold
Thresholds (in std. deviation units) which can be applied to the average image created from a group.
Marker radius
Radius used for image locations when image numbers are not displayed.

Show Image Numbers
A toggle to specify whether image number of only locations are shown on the map.
