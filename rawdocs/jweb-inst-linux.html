• GENERAL

    JWeb is a point-and-click interface for display and interaction with SPIDER images and data written by Dr. Bimal Rath. JWeb is written in Java and was developed and tested on Linux and Windows computers.

  • JWeb INSTALLATION

    1. Download and install the SPIDER/Web distribution and place it in an appropriate directory: PARENT-DIR.

      The following JWeb related files will be created under: PARENT-DIR/web/jweb

      File Contents
      src directory JWeb source code
      linux/j2re-1_4_2_06-linux-i586.bin JRE install script
      linux/jai-1_1_2_01-lib-linux-i586-jre.bin JAI install script
      linux/linux.WEB.jar Linux JWeb Java archive
      linux/font.properties Linux JWeb fonts
      linux/jweb.cshrc JWeb 'C' shell environment variables

    2. Most Linux distributions install a Java Run Time Environment (JRE) by default. However only a few Linux distributions install the Java Advanced Imaging Run Time Environment (JAI) which is needed by JWeb. You could add JAI to your Java installation but due to various issues it is often simpler to install a second JRE containing JAI as described here.

      1. Access your JWeb directory:
        cd spider/web/jweb/linux

      2. Install the JRE included with JWeb by executing:
        j2re-1_4_2_06-linux-i586.bin

      3. Install JAI in the new Java directory:
        cd j2re1.4.2_06
        ../jai-1_1_2_01-lib-linux-i586-jre.bin

      4. Return to the JWeb directory:
        cd ..

    3. Set necessary environment variables and alias for JWeb usage under Java.
      If you are using the 'C' shell, edit the first non-comment line in: jweb.cshrc and create a $JWEB_DIR environment variable specifying location for JWeb on your system i.e.:
      setenv   JWEB_DIR   Enter_path_of_your_JWeb_directory_here
      Then place the edited contents of jweb.cshrc in the .cshrc startup file in your home directory.

    4. The previous step will also create alias, 'jweb', for starting JWeb with a specified maximum amount -Xmx512m of memory allowed. You may adjust this amount to suit your computer.

    5. If you are not using a 'C' shell then you will have to alter the last two steps as appropriate for your shell.

  • JWeb TESTING

    • Information on the use of JWeb is given in the JWeb manual.

    • Test JWeb :
      • jweb
      USE MOUSE TO SELECT MENU ITEMS FROM PULL-DOWN MENUS

    • If you get error messages like:
      • libXp.so.6: Cannot open shared object file: No such file or directory
      libXtst.so.6: Cannot open shared object file: No such file or directory
      You may have a 64 bit OS which lacks some 32 bit X Window libraries which you will need to install. For distributions which use apt-get this can be done:
      sudo apt-get install ibxp6:i386
      sudo apt-get install libxtst6:i386

    • If you get error messages like:
      • Exception in thread "GtkMain" java.lang.UnsatisfiedLinkError: gtkInit ... ....
      You are not using the Java run time environment packaged with JWeb. The GNU environment that you are trying to use may be defective.
